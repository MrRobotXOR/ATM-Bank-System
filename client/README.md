# Bank Management System

A full-stack Bank Management System built with React, Vite, Node.js, Express, and MySQL.

## Features

- User Registration & Login
- JWT Authentication
- Account Management
- Deposit & Withdraw Money
- Transaction History
- Admin Dashboard
- User Management

## Tech Stack

### Frontend
- React
- Vite
- React Router DOM
- Axios

### Backend
- Node.js
- Express.js
- MySQL
- JWT
- Bcrypt

### Frontend
